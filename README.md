# Video Game Sales Prediction Engine
A Python program that makes video game sales predictions. You can enter the attributes of a game (year, user and critic ratings, genre, and ESRB rating) and it will create a prediction using Scikit-Learn's RandomForestRegressor.

Dataset Credit:
https://www.kaggle.com/rush4ratio/video-game-sales-with-ratings

Other features:
* GUI built in Qt/PyQt
* Ability to retrain model in the GUI
* RSA Encrypt/Decrypt a csv dataset using Crypography from the GUI
* View different charts created with Pandas/Matplotlib
